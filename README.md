# Fable Works

Claude Code の **Fable 5** で作った小さな作品を残していく個人開発シリーズ。
実用性より「こんなものが作れるのか」という驚きを優先する。

## 作品一覧

| # | 作品 | 概要 | デモ |
|---|------|------|------|
| 01 | [Generative Art](01-generative-art/) | 単一HTML・依存ゼロの生成アート。フローフィールド / Boids / 重力銀河 / 太陽系の4モード、タッチ干渉つき | [開く](https://shuyu0514.github.io/fable-works/01-generative-art/) |

## ルール

- 各作品は番号つきフォルダ（`01-xxx/`）で独立させる
- 依存は最小限に。単一ファイルで動くものを良しとする
- Made with Claude Code (Fable 5)
